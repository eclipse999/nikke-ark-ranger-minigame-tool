# NIKKE Ark Ranger Minigame Tool

繁體中文 | [English](README.en.md)

背包最佳化工具：為手遊《勝利女神：NIKKE》Ark Ranger 小遊戲設計的 9×9 背包空間規劃器。

輸入棋盤可用區域與道具數量，求解器自動計算最高利用率的擺放方案。

> 本工具參考並重製自 [nikke-mini-game.netlify.app](https://nikke-mini-game.netlify.app/)，保留其三欄式操作介面，並新增**道具旋轉**支援。

## 線上版本

已部署到 Netlify，可直接使用：

- [繁體中文](https://nikke-ark-ranger-minigame-tool.netlify.app/)
- [English](https://nikke-ark-ranger-minigame-tool.netlify.app/?lang=en)

## 功能

- **9×9 背包棋盤** — 點擊格子切換可用／不可用，支援重置及一鍵全開
- **P01–P15 完整道具** — 全部 15 種道具，自動產生 0°/90°/180°/270° 旋轉形狀，去除重複
- **截圖半自動匯入** — 可匯入遊戲完整截圖，於瀏覽器本機推估上方 9×9 背包格狀態，再手動校正
- **回溯求解器** — 優先擺放大面積道具，含面積剪枝與時間上限（預設 1 秒）
- **多候選解法** — 保留最多 3 組最佳方案，可在結果區切換檢視
- **雙語介面** — 繁體中文 / English，支援 URL 參數 `?lang=en` 指定預設語言，切換後即時生效
- **本機運行** — 純前端，無需後端，資料不離開瀏覽器

## 快速開始

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

開啟瀏覽器前往 `http://localhost:5173`。

## 使用方式

1. **設定棋盤** — 點擊灰色格子切換為不可用（深色），或按「全部可用」／「重置背包」
2. **輸入數量** — 在中央道具清單填入每種道具的持有數量
3. **開始計算** — 點擊「開始計算」，求解器會在 1 秒內找出最佳擺法
4. **檢視結果** — 右側顯示可用格、填入格與利用率，棋盤上以不同顏色標示各道具位置；可用箭頭切換候選解法
5. **切換語言** — 點擊右上角語言按鈕

截圖匯入為半自動輔助功能，只分析瀏覽器本機圖片，不會上傳。匯入後請確認棋盤格狀態，必要時手動修正。

```

├── items.ts        # P01–P15 道具定義與旋轉形狀產生

├── i18n.ts         # 繁體中文 / English 文案
├── App.tsx         # 主元件：三欄式 UI、求解狀態管理
├── main.tsx        # React 進入點
└── styles.css      # 全域樣式
```

### 求解策略

- 回溯搜尋 + 貪心啟發：優先嘗試面積大的道具
- 對每種道具嘗試所有去重旋轉形狀，左到右、上到下掃描可放置位置
- 面積剪枝：若剩餘可用格小於剩餘道具最小面積，提前回溯
- 時間上限 1 秒：超時回傳目前最佳解，標記「時間內最佳」
